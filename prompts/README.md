# System Prompts for AI Studio

This directory contains reusable system prompts for AI Studio integration.

## 📁 Contents

### `hadith-content-studio-v3.0.md`
Complete system prompt for the Hadith Content Studio - an integrated pipeline for modernizing classical Islamic hadith texts and preparing them for professional TTS voice narration.

**Features:**
- Two-stage workflow (word modernization + voice script generation)
- Smart word detection with risk assessment
- Interactive user approval process
- Multiple output formats (video, shorts, minimal)
- Prosody and directive system for ElevenLabs TTS

**Target Audience**: "İyilerin Yolu" YouTube Channel

### `USAGE.md`
Comprehensive usage guide for implementing Hadith Content Studio v3.0 in AI Studio.

**Includes:**
- Step-by-step setup instructions
- Command reference
- Model configuration recommendations
- Protected terms list
- Best practices
- Troubleshooting guide

## 🚀 Quick Start

1. Copy the contents of `hadith-content-studio-v3.0.md`
2. Open AI Studio → System Instructions
3. Paste the prompt
4. Save and start using

For detailed instructions, see `USAGE.md`

## 📋 Supported Workflows

- **Word Modernization**: Turkish archaic → modern terminology
- **Voice Scripting**: TTS-ready narrative scenarios
- **Format Options**: Full video, short-form, minimal
- **Batch Processing**: Handle multiple hadiths sequentially
