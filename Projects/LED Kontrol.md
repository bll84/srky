# LED Kontrol Projesi

## Aciklama
Raspberry Pi GPIO pinleri kullanarak LED kontrolu yapan Python scripti.

## Donanim
- Raspberry Pi
- LED (GPIO pin 17)
- 330 Ohm direnc

## Dosyalar
- [[led_control.py]] - Ana kontrol scripti

## Notlar
- GPIO.BCM modu kullaniliyor
- LED 5 kez yanip sonuyor (1 saniye aralikla)
- `Ctrl+C` ile guvenli cikis destekleniyor
